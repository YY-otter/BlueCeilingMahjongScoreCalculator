# BlueCeilingMahjongScoreCalculator
青天井で計算する麻雀点数計算ページです<br>
大抵のブラウザでは四倍役満(50翻超)ぐらいの和了から桁が全部表示されないかもしれません<br>
なお、ES2020に準拠しているブラウザでないと動作しません、IEは厳しいかも

現在対応している計算方法は四麻と、三麻の「ツモ損」「北家分折半」「折半」「親3倍被り」の5種類です
